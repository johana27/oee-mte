Información del Reporte

El reporte incluye:

  • Datos de Recaps Diarios: Piezas planeadas/producidas, defectos totales, tiempo muerto, disponibilidad, rendimiento, calidad y OEE
  • Registros de Downtime: Causa, inicio, fin, duración y comentarios de cada tiempo muerto
  • Registros de Defectos: Causa, cantidad, tipo, modelo afectado y comentarios

Tipos de reporte:

  • Semanal: Datos de una semana específica del mes seleccionado (7 días)
  • Mensual: Datos consolidados de todo el mes seleccionado
  • Trimestral: Últimos 3 meses hasta el mes seleccionado
  • Semestral: Primer semestre (Enero-Junio) o Segundo semestre (Julio-Diciembre) según el mes seleccionado
  • Anual: Datos consolidados de todo el año seleccionado

Nota: El reporte genera un archivo Excel con 22 columnas que incluyen todos los datos de producción, downtimes y defectos de la celda seleccionada en el periodo indicado.
